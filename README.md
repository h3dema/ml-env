# ml-env
Environment with machine learning tools


## Install with Docker

```
apt install -y git
docker build -t h3dema/ml https://github.com/h3dema/ml-env.git
docker run -it h3dema/ml
```

## Install locally

```
apt install -y git
git clone https://github.com/h3dema/ml-env.git
cd ml-env
bash ./sh.install
```
